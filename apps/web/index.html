Principer

Byggt för att hålla.

  • Säkerhet som grundkrav. Inte ett tillval i efterhand.
  • Integritet. Så lite data som möjligt, så tydligt hanterad som möjligt.
  • Tillgänglighet. Produkterna ska fungera för fler, inte färre.
  • Prestanda. Snabbt är en egenskap, inte en optimering.
  • Begriplig teknik. Val som går att förklara går att förvalta.
